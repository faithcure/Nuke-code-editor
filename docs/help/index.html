Overview

Built for Nuke scripting workflows

Nuke Code Editor is a PySide2-based IDE embedded into Foundry Nuke. It brings a multi-tab editor, output console, workspace tools, and Nuke-specific helpers into a single environment.

Multi-tab Editing

Split views, tab management, and fast file switching.

Nuke-aware Tools

Node creation helpers, inline suggestions, and completions.

Output + Console

Live execution results and quick debugging feedback.

Install

Installation and Launch

Follow the recommended setup from README.md:

import nuke, os
nuke.pluginAddPath(os.path.join(os.path.dirname(__file__), "CodeEditor_v02"))
  • Restart Nuke after adding the path.
  • Open the editor from the Nuke menu.

Support

Troubleshooting

Editor does not open

Verify that the plugin path is added correctly and restart Nuke.

Settings file missing

Delete the settings file and relaunch to recreate defaults.

Node Creator list is empty

Check that Nuke plugin paths are valid and readable.

GitHub actions fail

Verify credentials and network access in Settings.

Data

Paths and Data Locations

  • User data is stored in a per-user app data folder.
  • Cache files are stored under the user cache path.
  • Example settings file: editor/settings/settings.json.example.
